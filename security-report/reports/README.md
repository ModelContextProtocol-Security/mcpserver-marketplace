## Report Types

### Landscape Reports
High-level snapshots of the ecosystem (marketplace counts, types, trends).

### Analysis Reports
Deep dives into specific aspects (client marketplace integration, security patterns).

### Process Reports
Meta-analysis of evaluation methodology and findings.

## How Reports Are Generated

1. **Data collection**: Using prompts in `../prompts/`
2. **Evaluations**: Individual assessments in `../evaluations/`
3. **Synthesis**: Aggregating findings into reports here

## When to Create a Report

- After completing a significant batch of evaluations
- When identifying cross-cutting patterns
- When the ecosystem snapshot needs updating
- For specific research questions
