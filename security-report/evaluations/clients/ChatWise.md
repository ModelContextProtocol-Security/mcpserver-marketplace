# ChatWise

## Notes

Desktop AI chatbot (macOS/Windows) supporting multiple LLMs (ChatGPT, DeepSeek, Claude, Gemini, Llama). Local data storage for privacy. MCP via Settings > Tools > Add with manual config or JSON import from other apps. Supports mcp-add URL scheme for one-click installation links. No built-in marketplace for browsing servers.
