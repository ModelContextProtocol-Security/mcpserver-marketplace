# Augment Code

## Notes

AI coding assistant with built-in "Easy MCP" - curated one-click integrations for popular tools (CircleCI, MongoDB, Redis etc). Users can also manually configure servers via settings panel or JSON import. Supports both local and remote (HTTP/SSE) MCP servers.
