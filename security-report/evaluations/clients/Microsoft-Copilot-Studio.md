# Microsoft Copilot Studio

## Notes

Microsoft's enterprise agent builder with MCP support. Built-in MCP servers catalog. MCP onboarding wizard in UI. Connector infrastructure provides VNet integration, DLP controls, OAuth/API key auth. Streamable transport (SSE deprecated Aug 2025). Dynamic tool discovery and updates from servers.
