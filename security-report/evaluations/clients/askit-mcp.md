# askit-mcp

## Notes

Python asyncio library and CLI for MCP. Configuration through mcp_config.json file supporting local (STDIO) and remote (HTTP/SSE) servers. No marketplace features - users are pointed to external resources (MCP Servers repo, Awesome MCP Servers) for server discovery.
