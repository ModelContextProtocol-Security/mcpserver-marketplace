# Kiro

## Notes

AWS agentic AI IDE for spec-driven development. Native MCP integration including remote servers. Kiro Powers feature for dynamic MCP loading from partners (Datadog, Dynatrace, Neon, Netlify, Postman, Supabase). Powered by Claude Sonnet 4.0/3.7. Free during preview, then $19-39/month plans.
