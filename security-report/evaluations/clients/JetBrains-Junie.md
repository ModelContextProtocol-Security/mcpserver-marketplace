# JetBrains Junie

## Notes

JetBrains AI coding agent for IntelliJ IDEA, PyCharm, GoLand, PhpStorm, WebStorm, RubyMine, RustRover. MCP support via config files (~/.junie/mcp.json or project .junie/mcp). Integrated into AI Chat. 30% faster performance. No built-in marketplace - configure servers manually via JSON.
