# MCPJam

## Notes

MCP server testing/debugging platform. Visual inspector for tools, resources, prompts, OAuth. LLM playground with frontier models (GPT-5, Claude Sonnet, Gemini 2.5) - no API key needed. Supports STDIO, SSE, HTTP. OpenAI app emulator. Local-first development. Apache 2.0. Via npx or Docker.
