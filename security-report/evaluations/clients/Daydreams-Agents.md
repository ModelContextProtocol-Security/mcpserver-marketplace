# Daydreams Agents

## Notes

TypeScript framework for building commerce agents. Native MCP support via extension system. Features composable contexts, persistent memory, universal runtime (Node.js, browsers, Deno, Bun, edge). Connect to MCP servers via stdio or HTTP/SSE. No built-in marketplace - developers configure MCP connections programmatically.
