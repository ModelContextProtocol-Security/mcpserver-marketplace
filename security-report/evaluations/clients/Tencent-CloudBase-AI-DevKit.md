# Tencent CloudBase AI DevKit

## Notes

Bridge connecting AI IDEs (Cursor, Copilot, Codex) to Tencent CloudBase. MCP server provides file operations, function management, domain management, knowledge base search. One-click deployment. WeChat Mini-Program support. Chinese market focus. Node.js v18.15+. No marketplace.
