# HyperChat

## Notes

Open source chat client by BigSweetPotatoStudio. Built-in MCP Plugin Marketplace with one-click installation. Third-party MCP support via manual config (command, args, env). Multi-LLM support (OpenAI, Claude, Qwen, Deepseek, GLM, Ollama). Features: RAG, scheduled tasks, multi-session, agent management, WebDAV sync.
