# MCPHub

## Notes

Neovim plugin (mcphub.nvim) that brings MCP to Neovim. Features server discovery via MCP Registry browsing, manual and AI-assisted installation. Supports all transports (HTTP, SSE, STDIO), OAuth with PKCE. Integrates with popular Neovim chat plugins (Avante, CodeCompanion, CopilotChat). Real-time capability updates.
