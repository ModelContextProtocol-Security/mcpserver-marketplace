# Smithery Playground

## Notes

Smithery is THE MCP server registry/marketplace. Playground allows testing MCP servers in browser. CLI for install/uninstall/inspect servers. Hosted option where Smithery runs servers. Includes servers for Google Drive, Calendar, Todoist, Instagram, databases, web search, etc. Client-agnostic.
