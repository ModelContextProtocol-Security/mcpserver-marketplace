# mcp-use

## Notes

Python/TypeScript library by Pietro Zullo to connect any LLM to any MCP server. LangChain integration for tool calling models (OpenAI, Anthropic, Groq, Llama). MCPAgent, MCPClient components. Supports HTTP, WebSocket, stdio transports. Session management. 6 lines of code to build agents. No marketplace.
