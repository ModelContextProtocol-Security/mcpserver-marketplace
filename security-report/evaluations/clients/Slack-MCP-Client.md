# Slack MCP Client

## Notes

Go-based Slack bot by tuannvm bridging Slack and MCP servers. Universal MCP compatibility (HTTP, SSE, stdio). Multi-provider LLM (GPT-4.1, Claude 4.5, Ollama). Agent Mode with LangChain for multi-step reasoning. Custom LLM-MCP bridge layer. Helm chart for K8s. MIT licensed. No marketplace.
