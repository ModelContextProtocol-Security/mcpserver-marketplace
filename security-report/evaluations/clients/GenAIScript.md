# GenAIScript

## Notes

Microsoft's scripting tool that implements both MCP client and server. Scripts become MCP tools automatically. Configure MCP servers via script metadata or defTool using Claude config syntax. VS Code Copilot integration. Features prompt injection detection, tool signatures for security. No built-in marketplace - configuration via code/JSON.
