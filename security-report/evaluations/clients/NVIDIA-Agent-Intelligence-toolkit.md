# NVIDIA Agent Intelligence toolkit

## Notes

NVIDIA AIQ Toolkit (v1.1.0) with full MCP support. MCP client (mcp_tool_wrapper for remote tools) and MCP server (aiq mcp command). Framework-agnostic - works with LangChain, LlamaIndex, CrewAI, Semantic Kernel. Formerly AgentIQ. pip install aiqtoolkit. No marketplace - config based.
