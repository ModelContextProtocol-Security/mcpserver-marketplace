# eechat

## Notes

Open-source local AI chat app (Windows/macOS/Linux) with extensive MCP support. MCP management interface for visual install/start/stop/remove of AI tools. Form-based adding with AI-powered config or README parsing. Multiple LLM providers (OpenAI, Anthropic, Groq). All data stored locally for privacy.
