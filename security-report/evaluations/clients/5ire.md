# 5ire

## Notes

5ire is a cross-platform AI assistant that functions as an MCP client. It integrates with MCPSvr, a companion MCP server registry also maintained by the same developer. Users can browse and install servers directly from within the app.

Key marketplace features:
- Built-in server browser connected to mcpsvr.com
- One-click installation from the registry
- Developer can add "one-click server installation" buttons to their own websites that integrate with 5ire
