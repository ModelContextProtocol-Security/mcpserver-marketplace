# Genkit

## Notes

Google/Firebase open-source AI framework (JavaScript, Go, Python). MCP plugin provides client (createMcpHost/createMcpClient) and server (createMcpServer) capabilities. Consume MCP tools/prompts/resources or expose Genkit tools as MCP servers. Works with Google AI, OpenAI, Claude, Ollama. No built-in marketplace - code-based configuration.
