# Cursor

## Notes

AI code editor with built-in MCP marketplace. Features include: browsable server directory, one-click installation with OAuth support, curated popular servers. External directories (cursor.directory, mcpmarket.com) also provide Cursor-specific servers. Config via mcp.json or settings UI. Supports stdio, SSE, and Streamable HTTP transports.
