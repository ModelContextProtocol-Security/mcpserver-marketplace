# oterm

## Notes

Terminal client for Ollama by ggozad. MCP tools, prompts, and sampling support. Acts as gateway between Ollama and MCP servers. SSE and WebSocket transports for MCP. Persistent chats in sqlite. Works on Linux/macOS/Windows. Install via uvx oterm. MIT licensed. No marketplace - manual MCP config.
