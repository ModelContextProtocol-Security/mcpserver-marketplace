# Superinterface

## Notes

AI infrastructure platform for building in-app AI assistants. React components and Hooks. MCP generally available - connect to external servers. Supergateway (open source) converts stdio MCP to SSE. Supermachine provides hosted MCP servers. Works with Cursor, Zed, Claude Desktop, Librechat.
