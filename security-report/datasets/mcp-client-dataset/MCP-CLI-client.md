# MCP CLI client

## Notes

Python CLI for running LLM prompts with MCP by adhikasp. Alternative to Claude Desktop. Config at ~/.llm/config.json supporting multiple providers (OpenAI, Groq, local llama). Features: clipboard support, multimodal inputs (images), prompt templates, tool confirmation options. MIT licensed. No marketplace.
