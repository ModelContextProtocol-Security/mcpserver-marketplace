# MCP SuperAssistant

## Notes

Browser extension (Chrome/Firefox) bringing 6000+ MCP servers to AI platforms. Supports ChatGPT, Perplexity, Gemini, Grok, Google AI Studio, OpenRouter, DeepSeek, Kagi, T3 Chat, GitHub Copilot, Mistral AI. Detects/executes MCP tool calls, inserts results back. Plugin architecture. Proxy server for remote MCPs.
