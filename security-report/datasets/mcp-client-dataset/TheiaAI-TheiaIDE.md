# TheiaAI/TheiaIDE

## Notes

Eclipse open source IDE framework with MCP integration. Built-in MCP-powered agents: GitHub agent, App Tester (Playwright). Connect to third-party MCP servers (Git, databases, web search). Multi-LLM support (OpenAI, Anthropic, Azure, Ollama). Discussion ongoing for MCP server marketplace with dynamic installation.
