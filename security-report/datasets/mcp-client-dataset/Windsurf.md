# Windsurf

## Notes

AI code editor (formerly Codeium). Has built-in Plugin Store - curated MCP marketplace with one-click installation. Official plugins marked with blue checkmark. Supports both local (stdio) and remote (http) servers. Enterprise features: team admins can toggle MCP access and whitelist approved servers. Config via mcp_config.json (Claude Desktop compatible). 100-tool limit per session.
