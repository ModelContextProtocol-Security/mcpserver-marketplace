# kibitz

## Notes

Free open-source Replit alternative for daily coding work. Chat interface for Anthropic/OpenAI APIs with MCP support. Uses ws-mcp (WebSocket wrapper for MCP stdio servers). Configure per-project settings. Available as mobile app and desktop (Windows/macOS/Linux). MIT licensed. No built-in marketplace.
