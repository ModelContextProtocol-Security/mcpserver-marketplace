# Roo Code

## Notes

VS Code extension (21k+ stars) with built-in Roo Code Marketplace for discovering and installing MCPs and Modes. Features curated lists, single-click install, project/global scope options. Recommends Context7 as default MCP server. Multiple operating modes: Code, Architect, Ask, Debug, Custom. Community-driven extensions.
