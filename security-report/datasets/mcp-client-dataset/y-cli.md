# y-cli

## Notes

Tiny terminal chat app by luohy15 with MCP client support. Multiple bot configs (any base_url/api_key/model). MCP integration (stdio/SSE) with persistent daemon. Reasoning model support (Deepseek-r1, o3-mini). Chat data in JSONL files. Also has y-gui web version. Listed on official MCP clients. No marketplace.
