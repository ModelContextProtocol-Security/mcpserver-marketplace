# Warp

## Notes

AI-powered terminal with built-in MCP server sharing. Features:
- **Shared MCP Servers**: Out-of-the-box servers anyone can install from Shared section
- **Team Sharing**: Share configs with teammates (sensitive env vars auto-scrubbed)
- **OAuth Login**: One-click installation for compatible servers
- **Centralized UI**: MCP Servers page via Warp Drive/Command Palette
- **Cloud Sync**: Servers synced to Warp Drive, auto-start option

Config via JSON snippet or UI. Supports CLI (command) and SSE (URL) server types.
