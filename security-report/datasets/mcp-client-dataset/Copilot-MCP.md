# Copilot-MCP

## Notes

VS Code extension by Vikash Loomba enabling MCP servers with GitHub Copilot Chat. Features Server Discovery panel for finding/installing servers. Cloud MCP integration for remote OAuth-based servers. Supports process-based and SSE servers. Auto-reconnect for resilient connections. GPL v3 licensed.
