# fast-agent

## Notes

MCP-native Python framework for building agents and workflows by evalstate. Declarative syntax for composing prompts and MCP servers. Supports Anthropic, OpenAI, Google, Azure, Ollama, DeepSeek. Features OAuth, Streamable HTTP inspection, multi-modal (images/PDFs). Can host agents as MCP servers. No built-in marketplace.
