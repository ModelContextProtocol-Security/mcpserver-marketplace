# Chorus

## Notes

Mac desktop app for chatting with multiple AI models simultaneously (o3, Claude, Gemini, DeepSeek). Built on Tauri/Rust. MCP server support via Settings > Connections - add local or remote MCPs. Built-in tools for web search, terminal, GitHub. No marketplace - users add custom MCP servers manually.
