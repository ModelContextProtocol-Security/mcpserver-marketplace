# SpinAI

## Notes

TypeScript AI agent framework with Smithery MCP registry integration. Install any Smithery MCP via npx spinai-mcp. Converts MCPs to SpinAI Actions - "whole toolbox without writing code". createActionsFromMcpConfig function. Works with OpenAI, Vercel AI SDK models.
