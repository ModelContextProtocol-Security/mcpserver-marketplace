# ChatGPT

## Notes

OpenAI's ChatGPT with full MCP support in Developer Mode (beta). Add MCP connectors via Settings > Connectors > Advanced > Developer Mode. Supports SSE and streaming HTTP with OAuth. Available to paid plans (Pro, Plus, Business, Enterprise, Education). Remote servers only - no localhost. OpenAI joined MCP steering committee and Agentic AI Foundation.
