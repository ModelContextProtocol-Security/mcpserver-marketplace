# Nerve

## Notes

Agent Development Kit (ADK) by evilsocket with full MCP integration (v1.5.0). First framework to define MCP servers in YAML. Acts as both MCP client and server. Access publicly available MCP servers. Built on LiteLLM (OpenAI, Anthropic, Ollama). Expose agents as tools. GPL 3 licensed.
