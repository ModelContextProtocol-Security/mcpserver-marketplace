# Dolphin-MCP

## Notes

Python multi-server bridging MCP client by cognitivecomputations. Connects multiple MCP servers to any LLM (OpenAI, Anthropic, Ollama, LMStudio). CLI with interactive mode and library API. Config via mcp_config.json. Automatic tool discovery from connected servers. No marketplace - manual configuration required.
