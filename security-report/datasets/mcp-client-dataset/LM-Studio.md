# LM Studio

## Notes

Local LLM runner with MCP Host support (v0.3.17+). Config via mcp.json file (Cursor-compatible format) or deep links. No built-in marketplace - users directed to awesome-mcp-servers repo and community for server discovery. Supports local and remote MCP servers. Security warning about untrusted MCPs.
