## Notes

CLI host enabling LLM/MCP tool interactions by vincent-pli. Supports OpenAI, Azure OpenAI, Deepseek, Ollama models. Config at ~/.mcp.json (Claude Desktop format). Interactive commands: /tools, /resources, /exclude_tool. Message window context management. Apache 2.0 licensed. No marketplace.
