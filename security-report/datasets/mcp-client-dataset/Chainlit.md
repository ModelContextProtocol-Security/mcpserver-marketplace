# Chainlit

## Notes

Python platform for building AI applications with built-in MCP support. Configure MCP via on_mcp_connect handler and config.toml. Supports SSE, Streamable HTTP, and stdio connections. No built-in marketplace - developers manually configure MCP server connections.
