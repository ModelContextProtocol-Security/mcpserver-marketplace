# ChatMCP

## Notes

Cross-platform (macOS, Windows, Linux, iOS, Android, web) AI chat client with built-in MCP Server Market. Flutter-based. Supports OpenAI, Claude, Ollama, DeepSeek. Users can browse and install servers from marketplace. Community-driven server submissions via GitHub PRs.
