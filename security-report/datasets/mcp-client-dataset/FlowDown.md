# FlowDown

## Notes

Open source AI chat client for Apple platforms (iOS/macOS) by Lakr233. Swift/UIKit native app. MCP server support with .fdmcp file format for easy installation. Supports local models (Apple Silicon) and remote OpenAI-compatible providers. iCloud sync. Privacy-focused. AGPL-3.0 licensed. No built-in marketplace.
