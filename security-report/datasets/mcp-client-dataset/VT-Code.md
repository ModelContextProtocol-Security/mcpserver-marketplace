# VT Code

## Notes

Rust-based terminal coding agent with semantic code intelligence (Tree-sitter). MCP integration with configurable allow/deny/prompt policies. Multi-LLM support (OpenAI, Anthropic, Google, DeepSeek, Ollama, LM Studio). VS Code extension available. Zed integration via ACP. No marketplace - config based.
