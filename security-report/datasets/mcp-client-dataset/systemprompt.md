## Notes

First voice-controlled iOS/Android MCP client. Manage multiple MCP servers from single app. Built-in GitHub, Sentry, Reddit integrations. Connect to third-party MCP servers (Shopify, MongoDB). MCP OAuth support. Voice-controlled commands via AI/NLP. For developers and DevOps teams.
