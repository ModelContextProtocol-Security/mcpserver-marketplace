# Goose

## Notes

Open source AI agent from Block (23k+ stars). Has built-in extensions directory for discovering and installing MCP servers. Install via directory browser, CLI (`goose configure`), or UI. Built-in extensions: Developer (default), Computer Controller, Memory. Supports remote (SSE/HTTP) and command-line extensions. Config via ~/.config/goose/config.yaml. Auto-checks external extensions for malware before activation.
