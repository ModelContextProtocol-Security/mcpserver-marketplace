# mcp-agent

## Notes

Python framework by lastmile-ai for building AI agents with MCP. Implements Anthropic's "Building Effective Agents" patterns and OpenAI Swarm. Model-agnostic (OpenAI, Anthropic). Composable workflows as AugmentedLLM objects. Handles LLM mechanics, human input, persistent state. No marketplace.
