# Tutorial and Documentation Sites

## Definition

Educational content that includes MCP server installation instructions. Users follow step-by-step guides that often include copy-paste commands.

## Characteristics

- Step-by-step instructions
- Often includes copy-paste commands
- May be outdated
- No ongoing verification
- Author may not be security-aware

## Trust Model

- Educational, not authoritative
- Users expected to evaluate
- Author reputation-based

## Security Opportunities

- Can include security warnings
- Can link to verification sources
- Can keep content updated

## Security Risks

- **Copy-paste instructions bypass review**
- Outdated content with vulnerable servers
- AI agents may follow instructions literally
- No accountability for security issues

## Evaluation Priority

**Medium-High** - High trust from users seeking help, copy-paste behavior increases risk, AI agents may execute instructions without verification.

## Known Tutorial Sources

- Towards Data Science
- DataCamp
- Microsoft .NET Blog
- Northflank
- Medium
- The New Stack
- Simplescraper
- Apidog

Many tutorials with copy-paste installation commands. Users may not verify security before following instructions. AI agents may scrape these as authoritative sources.
